Politico

Vote and be Voted

Politico is a web-based application that enables you to create vote, be voted for and view the results in real time. You can participate live on your mobile phones, tablets and desktops.
How it Works
There are three navigation areas in the app: ➊ Welcome Page ➋ Contest Navigation to navigate within a contest Home: Returns you to home screen Voting: Voting screen to vote Results: Current results in real-time ➌ buttons to navigate between categories on the Voting and Results screens.
